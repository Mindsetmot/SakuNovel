# 📚 SakuNovel

SakuNovel adalah web pembaca Light Novel dengan tampilan modern, ringan, dan nyaman digunakan di perangkat mobile maupun desktop.

Website ini dibuat untuk menghadirkan pengalaman membaca novel yang simpel, fokus, dan immersive dengan nuansa dark fantasy serta accent ungu khas SakuNovel ✨

---

## ✨ Fitur

- 📖 Reader Light Novel modern
- 📚 Sistem volume & chapter
- 🌌 Tampilan dark mode aesthetic
- ⚡ Ringan dan cepat
- 📱 Responsive untuk mobile
- 📝 Support format Markdown untuk isi cerita
- 🔮 UI bertema fantasy-anime

---

## 🚧 Status

Project masih dalam tahap pengembangan aktif dan akan terus mendapatkan update serta penyesuaian fitur di masa depan.

---

## ⚠️ Catatan

Repository ini hanya berisi bagian frontend website.  
Backend, database, dan sistem admin dipisahkan dari repository publik.
